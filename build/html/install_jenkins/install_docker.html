Cài đặt Jenkins với Docker

Để cài đặt Jenkins bằng Docker, bạn có thể sử dụng lệnh sau:

Câu lệnh cài đặt
1docker run -d -p 8080:8080 -p 50000:50000 jenkins/jenkins:lts
