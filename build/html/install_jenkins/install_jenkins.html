Cài đặt Jenkins

  • Jenkins là một công cụ tự động hóa mã nguồn mở, giúp dễ dàng xây dựng và triển khai phần mềm.

  • Nó hỗ trợ nhiều plugin để tích hợp với các công cụ khác nhau và có thể tùy chỉnh cao để phù hợp với các quy trình phát triển.

  • Tuy nhiên nhược điểm là cần có server. Ngôn ngữ Jenkins sử dụng là Groovy có syntax rất phức tạp so với Gitlab-CI và Github Action sử dụng YAML sẽ dễ hiểu, dễ đọc hơn.
