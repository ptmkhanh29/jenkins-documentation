Cài đặt Jenkins trên Linux

Để cài đặt Jenkins trên Linux, bạn cần làm theo các bước sau:

Câu lệnh cài đặt
1sudo apt update
2sudo apt install openjdk-11-jdk
3wget -q -O - https://pkg.jenkins.io/debian/jenkins.io.key | sudo apt-key add -
4echo deb http://pkg.jenkins.io/debian-stable binary/ > /etc/apt/sources.list.d/jenkins.list
5sudo apt update
6sudo apt install jenkins
